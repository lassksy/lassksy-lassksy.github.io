hexo搭建博客后图片不显示的解决方法
kuroko Lv1
  2022-08-10 12:10:45 2022-08-10 12:10  
  • hexo 
 

本文将介绍在搭建博客后图片不显示两种解决方法

1.使用微博图床上传本地图片至网络

chrome扩展商店 https://chrome.google.com/webstore/detail/%E5%BE%AE%E5%8D%9A%E5%9B%BE%E5%BA%8A/pinjkilghdfhnkibhcangnpmcpdpmehk?hl=zh-CN

缺点

不支持svg格式文件

2.使用PicX上传本地图片至网络

在线使用入口 https://picx.xpoet.cn

只需选择一个 GitHub 仓库作为图床，然后在 PicX 官网 完成 Token 绑定和相应配置就能使用了。

官方文档 >> https://picx-docs.xpoet.cn

通过阅读官方文档的快速开始教程，可帮助你迅速上手 PicX 图床 <https://picx-docs.xpoet.cn/tutorial/get-start.html

结语

至此，博客图片不显示的问题成功解决。
